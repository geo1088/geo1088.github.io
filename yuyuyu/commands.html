Music Commands

!musichelp [command]

Prints a basic list of commands, or info on a command if [command] is specified. Also links to the RhinoBot help page, which has more information about the music bot's commands.

!play <song link or song search>

Add a song to the queue, or add the first youtube result for the provided search text. Songs can be linked in several different formats, including YouTube links or direct music files.

!queue

Prints the bot's current queue in chat.

!np

Prints the currently playing song in the chat.

!skip

Vote to skip the current song. The owner (LordTakuro) can use this to skip a song automatically. The song will be skipped when either 4 people or 1/4 of the current listeners vote to skip, whichever is less.

!search [service] [number] <query>

Interactively search for a video to add to the queue. The bot will look up number videos and prompts the user to accept or deny each video. This command times out after 30 seconds, and has a hard limit of 10 max search items. [service] can be youtube, soundcloud, or yahoo (or their abbreviations yt, sc, yh). [number] is limited to 10 max.

Non-music commands

!help [command]

Gets help information about a topic or command.

!choose <option>, <option>[, <option> ...]

Chooses randomly from a list.

!info

This command is broken.

!link

Don't use this command.

!ping [-t]

Pings the bot. To measure the time it takes to send the response, add the -t flag.

!roll <roll> [more rolls ...] [-d] [-r]

Executes dice rolls in standard dice notation (e.g. 2d6). Use the -d flag for a more detailed response, or the -r flag to be PM'd a raw log of the roll(s).

!say <message ...>

Makes the bot quote you.
